## ANOTAÇÕES AULA 1

-> Bit é 0 ou 1

-> Byte = 8 bits\
    --> Porção mínima que pode representar informação\
    --> Geralmente letras emojis etc\

-> Exemplos de bytes\
    --> 1024 bytes = 1 KB\
        ---> 2 ¹⁰ = 1024\
    --> 1024 KB = 1 MB\
    --> 1024 MB = 1 GB\
    --> 1024 GB = 1 TB\
    --> 1024 TB = 1 PB\
    --> 1024 PB = 1 EB\
    --> 1024 EB = 1 ZB\
    --> 1024 ZB = 1 YB\

-> MB != Mb\
    --> Mb = Megabit\
    --> Armazenamento = Byte\
    --> Transmissão = bit\

-> Modem = realiza modulação de ondas

-> DNS = Sistema de nome de domínio\
    --> Armazém de domínios de sites\

-> Rotas levam computador até o servidor
